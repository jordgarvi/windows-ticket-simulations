# New User Account & Device Onboarding

## Purpose

Provide a structured first-line onboarding workflow for preparing new user accounts, departmental access, and workstation environments for new starters.

---

## Common Onboarding Requirements

New starter onboarding requests may include:

- User account creation  
- Temporary password configuration  
- Departmental shared resource access  
- Workstation preparation  
- Desktop resource setup  
- Initial onboarding guidance  
- Application access preparation  

---

## Pre-Onboarding Checks

Before provisioning begins:

- Verify onboarding request originates from HR or approved management channels  
- Confirm manager approval  
- Review department access requirements  
- Confirm onboarding timeline and start date  
- Verify device preparation requirements  

This helps ensure onboarding activities are completed securely and consistently.
